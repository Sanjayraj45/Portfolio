## 🚀 Deployment

### Deploy on Netlify (Recommended)
1. Connect this GitHub repo to [Netlify](https://netlify.com)
2. Click "Deploy"
3. Your portfolio goes live instantly

### Deploy on Vercel
1. Sign in to [Vercel](https://vercel.com)
2. Click "New Project" and import this GitHub repo
3. Click "Deploy"

### Deploy on GitHub Pages
1. Go to Settings → Pages
2. Set Source to `main` branch
3. Your site will be live at `https://github.com/Sanjayraj45/portfolio`
